# 🌐 Portfolio - Juan Jesús González Hernández

Portfolio personal desarrollado con HTML y CSS donde presento mis proyectos,
tecnologías y perfil como desarrollador junior.

## 🚀 Sobre el proyecto

Este portfolio ha sido diseñado y desarrollado desde cero utilizando:

- HTML5
- CSS3 (Flexbox y Grid)
- Estructuración semántica
- Buenas prácticas de maquetación

El objetivo es mostrar mis proyectos personales y mi stack tecnológico
de forma clara, moderna y profesional.
